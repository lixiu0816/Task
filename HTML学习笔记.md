# HTML 简介
### HTML 实例解析
![HTML实例解析](https://www.runoob.com/wp-content/uploads/2013/06/02A7DD95-22B4-4FB9-B994-DDB5393F7F03.jpg)
- <!DOCTYPE html>声明为HTML5文档
**注**：docytype生命是不区分大小写的。

- <html>元素是HTML页面的根元素
- <head>元素包含了文档的元数据，如`<meta charset="utf-8">`定义网页编码格式为utf-8.
**注**：目前在大部分浏览器中，直接输出中文会出现中文乱码的情况，这时候我们就需要在头部将字符声明为 UTF-8 或 GBK。

- <title>元素描述了文档的标题

- <body>元素包含了可见的网页内容

- <h1>元素定义了一个大标题

- <p>元素定义一个段落

### 文本标签
```
- <h1> to <h6>：定义HTML标题
- (h1最大，h6最小)
- <p>：定义一个段落
- <br>：定义简单的换行（单标签）
- <hr>：定义水平线（单标签）
- <strong>/<b>：加粗文本(strong强调语义，b仅样式)
- <em>/<i>：斜体文本（em强调语义，i仅样式）
- <!--这是一个注释-->：定义一个注释
- <sub>：定义一个下标(字体偏下)
- <sup>：定义一个上标
- <del>：定义删除字
- <ins>：定义插入字
```
### 链接与媒体标签
```
- <a href="网址">链接文字</a>：超链接
- <img src="图片路径" alt="替代文本">：插入图片（单标签）
- <audio src="音频路径"controls></audio>：插入音频
- <video src="视频路径"controls></video>：插入视频
```
### 表格标题
```
- <table>：定义表格
- <th>：定义表格的表头
- <tr>：定义表格的行
- <td>：定义表格单元
- <caption>：定义表格标题
```
### 列表标题
```
- <ol>：定义有序列表
- <ul>：定义无序列表
- <li>：定义列表项
- <dl>：定义列表
- <dt>：定义列表项目
- <dd>：自定义列表项目
```
